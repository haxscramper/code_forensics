Detailed Description

template<typename... Args>
struct dod::MultiStore< Args >

Collecting of several different storage types, used as a boilerplate reduction helper - instead of wrapping around multiple Store<Id, Val> fields in the class, adding helper methods to access them via add/at you can write a MultiStore<> and list all the required data in the template parameter list. Supports both interned and non-interned storage solutions.
